# 5. Longest Palindromic Substring

## Решение 1 (Брутфорс)
3 цикла
- 1й - начало подстроки
- 2й - конец подстроки
- 3й проверяет строку на палиндром

Сложность - O(n^3) + O(1)

## Решение 2 (DP)
Сложность O(n^2) + O(n^2)
## Решение 3 (Расширяемся относительно центрального палиндрома)
Сложность O(n^2) + O(1)
